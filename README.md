# NexusSuite Unity Integrations

Official [Unity](https://unity.com/) custom packages and Editor tools for the [NexusSuite](https://nexussuite.net) game development toolchain.

## Available Modules

### 🦴 Bone Weaver (2D Skeletal Animation)
Adds native Editor tools to generate fully configured GameObjects from `.bwr` exports.
* **Download:** [BoneWeaver_Runtime.unitypackage](./bone-weaver/BoneWeaver_Runtime.unitypackage)
* **Instructions:** Import the package via `Assets > Import Package > Custom Package`. Right-click any `.bwr` file in your Project window and select `Bone Weaver > Generate Character Prefab`.
